Travel Project

Revolutionizing travel experiences with community and AI-driven recommendations

About the Project

This project consists of two integrated apps: Travel SNS and Travel Recommendation App. It aims to enhance user travel experiences by combining community-driven features and AI-powered recommendations.

Key Features

  • Travel SNS: Share and discover travel experiences, collaborate with friends, and record travel journals.
  • Travel Recommendation App: Get AI-based travel suggestions, plan trips, and preview destinations with AR/VR.
  • Integrated Functions: Seamlessly combine community insights with data-driven recommendations.
